Description
First formal project submission to Udacity for the Full Stack Web Developer Nanodegree. This project is a simple website that shows my favorite movies and their associated trailers. This aims to demonstrate the following: 1. A foundation is core Python programming concepts. 2. An understanding of the the role a simple web server has in receiving requests, executing a block of code and generating a response. The application will be rendered in a web browser generated from a local directory.

Installation Instructions

$ git clone https://github.com/nxl904/ud_Movie_Trailer
$ cd ud_Movie_Trailer

Usage
After creating a local directory, ud_Movie_Trailer requires the following steps to successfully execute: 

$ From terminal in local directory enter "python entertainment_center.py"
$ Program will render in defaulted browser.
